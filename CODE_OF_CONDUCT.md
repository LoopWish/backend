# Code of Conduct

Dette prosjektet følger Contributor Covenant Code of Conduct.

## Vårt løfte

Vi forplikter oss til å gjøre deltakelse i prosjektet til en opplevelse fri for trakassering.

## Håndheving

Rapporter uakseptabel atferd via GitHub i dette repoet.

Basert på Contributor Covenant v2.1: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
