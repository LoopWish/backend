# Contributing to Loopwish Backend

Takk for at du vil bidra!

## Retningslinjer

- Følg [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
- Hold PR-er små og fokuserte.
- Legg til/oppdater tester når relevant.

## Lokale sjekker

```bash
go test ./...
golangci-lint run
```

## Pull Requests

- Fyll ut PR-templaten i `.github/pull_request_template.md`.
- Beskriv endring og hvorfor.
- Link til issue hvis relevant.
